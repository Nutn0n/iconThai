Setup & Usage

Step 1 Install

iconThai is published in npm and yarn, so you can easily download it, using your preferred package manager:

Using npm

npm install iconthai

Using yarn

yarn add iconthai

Alternatively, you can use either GitCDN or RawGit to import the default flavor of mini.css. Simply add one of the following references inside your HTML page's '' tag:

<link rel="stylesheet" href="https://cdn.rawgit.com/jukbot/iconThai/d34cd0b8/icon-thai.css">

You can also find iconThai on cdnjs, which you can use to include it in your projects. We strongly suggest you add the following line inside your HTML page's

<link rel="stylesheet" href="https://cdnjs.com/libraries/icon-thai.css">
